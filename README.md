# README

# 開発言語
* ruby 3.0.1
* gem rails 6.1.7

# 就職Termの技術
* devise
* AWS
* Ajaxを使ったコメント機能

# カリキュラム外の技術
* ransack

# 実行手順
```
$ git clone git@github.com:asaaxis/camplet.git
$ cd camplet
$ bundle install
$ rails db:create && rails db:migrate
$ rails s
```
